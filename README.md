# movie-recommender-system
A smart movie recommendation system that not only suggests similar movies but also automatically recommends sequels (Part 2, Part 3, etc.) when a user enter movie name.
**Movie Recommender System with Sequel Detection**

A smart movie recommendation system that suggests sequels (Part 2, Part 3, etc.) first, followed by similar movies using machine learning techniques.


**Overview**

This project is a content-based movie recommender system enhanced with sequel detection.

>>If a user searches for a movie (e.g., Part 1), the system:

--First recommends movies from the same series/franchise
--Then suggests similar movies based on content similarity


**Features**

Automatic sequel recommendation (Part 1 > Part 2, Part 3)
>> Smart movie search with close matching
>> Content-based filtering using NLP
>> Fast and efficient recommendations
>> Clean and structured dataset handling.

**Tech Stack**
Python 
Pandas & NumPy
Scikit-learn
TF-IDF Vectorizer
Nearest Neighbors
